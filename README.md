# SBCL用户手册
## SBCL version 1.2.4 (2014-9)
该文档是SBCL软件系统的一部分，见README文件以获取更多信息。

该文档大部分是从CMUCL系统的文档获取的。CMUCL系统是从卡耐基梅隆大学产生的并且之后被发布到公共领域。该文档处于公共领域，并且没有任何限制。见COPYING文件和CREDITS文件以获取更多信息。
